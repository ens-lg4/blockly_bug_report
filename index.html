Bug report #2 - compatibility rules not checked when removing a block from a stack of statements
The rules are defined as follows:
  • Green can be followed by Blue
  • Green can be followed by Purple
  • Blue can be followed by Green
Try making a stack of Blue-Green-Purple. Now use right-click to bring up the context menu on the Green block and delete it. The Blue-Purple will stick into an "illegal" pair.
